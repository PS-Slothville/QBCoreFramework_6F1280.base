Note:
Interior will work only on a newest FiveM Game Build 2372!

Update your server artifact to 4268, enable Canary version and use +set sv_enforceGameBuild tuner in your start command to enable it.

Start: 
start ipl_tunergarage
start int_tunergarage

You can change tuner garage style by enabling/disabling different styles in client.lua in ipl_tunergarage folder and then restart the resource
